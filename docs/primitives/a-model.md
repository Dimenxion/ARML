# `<a-model>`

The model primitive loads a model from Unity AssetBundle.

## Example

```html
<a-scene>
  <!-- Using the asset management system. -->
  <a-obj-model src="bundlename:assetname"></a-obj-model>
</a-scene>
```

## Attribute

| Attribute | Component Mapping | Default Value |
|-----------|-------------------|---------------|
| src       | model.src         | null          |
