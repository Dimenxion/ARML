# `<a-text>`

[text]: ../components/text.md

Wraps the [text component][text].

## Example

```html
<a-text value="Hello, World!" color="red"></a-text>
```

## Attributes

| Attribute      | Component Mapping  |
|----------------|--------------------|
| color          | text.color         |
| opacity        | text.opacity       |
| value          | text.value         |
