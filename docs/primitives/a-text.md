# `<a-text>`

[text]: ../components/text.md

Wraps the [text component][text].

## Example

```html
<a-text value="Hello, World!"></a-text>
```

## Attributes

| Attribute      | Component Mapping  |
|----------------|--------------------|
| color          | text.color         |
| font           | text.font          |
| opacity        | text.opacity       |
| transparent    | text.transparent   |
| value          | text.value         |
